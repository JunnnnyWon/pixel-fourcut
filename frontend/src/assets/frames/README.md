# Frames

This directory is reserved for production frame overlay assets used by the
print workflow.

Expected output files:

- `frame-01-signature-white.png`
- `frame-02-signature-black.png`
- `frame-03-colored-pop.png`
- `frame-04-festival-sticker.png`
- `frame-05-pixel-arcade.png`
- `frame-06-club-badge.png`

All frame assets should be:

- portrait `1200 x 1800`
- transparent PNG overlays
- designed for two stacked `3:2` photo slots
- suitable for Canon SELPHY postcard-size printing
